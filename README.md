# Binance

### Project setup

```sh
pnpm install
```

Create `.env` file based on the `.env.example` file with environment variables.

### Start development

```sh
pnpm dev
```

### Development tools

```sh
pnpm lint
pnpm typecheck
pnpm format
pnpm check # run all scripts above at once
```

### API documentation

Swagger documentation is available under the `http://localhost:3001/reference` URL on the development environment.
